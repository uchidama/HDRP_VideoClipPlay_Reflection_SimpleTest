Unity HDRP Video Clip Play Reflection Simple Test

# Memo
Reflection Probeだと、VideoPlayerが反射しないようなので、昔ながらの反射用のオブジェクトを作って半透明の床下に反転させて置く方法で、床面への反射を表現してみた。
